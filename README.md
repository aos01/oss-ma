1️⃣ Header clair
# tpl — Project Template Platform

Generate, enforce and maintain clean, transferable project architectures.

2️⃣ Le problème
## Why
Projects become hard to maintain when:
- teams change
- standards are implicit
- architecture decisions are undocumented

tpl solves this by combining templates, standards and enforceable policies.

3️⃣ Ce que fait tpl
## What tpl does
- Generate clean, opinionated project templates
- Enforce architecture and quality rules
- Apply company-wide policies via packs

4️⃣ Exemple rapide
npx tpl init react-ts my-app --pack company-pack-a
cd my-app
npx tpl check
